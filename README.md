# AI Loan Calculator

## Overview
AI Loan Calculator is a full-stack web application for calculating loan repayments, comparing scenarios, storing results securely, and generating AI-powered financial insights.

## Features
- User authentication
- Loan repayment calculator
- AI Loan Advisor (Groq via Supabase Edge Functions)
- Save and manage loan scenarios
- Charts and amortization schedule
- PDF export, printing, and sharing
- Responsive UI

## Tech Stack
- React + TypeScript + Vite
- Supabase (Auth, PostgreSQL, Edge Functions)
- Groq LLM
- Tailwind CSS
- Vercel

## Installation
```bash
npm install
npm run dev
```

## Environment Variables
- VITE_SUPABASE_URL
- VITE_SUPABASE_ANON_KEY

## Deployment
Frontend deployed on Vercel. Backend hosted on Supabase.
